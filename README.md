# Сайт-візитка

Мій персональний сайт-візитка, створений для презентації навичок, проектів та резюме.  
Проект розроблений як портфоліо для HR та рекрутерів, щоб швидко ознайомитися з моїм досвідом у сфері веб-розробки.

## Функціонал
- Фіксований хедер з навігацією по секціях
- Плавний скролінг до розділів
- Світла та темна тема (збереження вибору у `localStorage`)
- Секції:
  - **Про мене** — коротка інформація про досвід та мотивацію
  - **Навички** — згруповані теги (Frontend, Backend, Інструменти, Soft Skills)
  - **Проекти** — прев’ю та демо моїх робіт (ToDo App, Weather Dashboard)
  - **Резюме** — кнопки для завантаження та перегляду онлайн
  - **Контакти** — email, Telegram, GitHub, LinkedIn

## Використані технології
- **HTML5, CSS3, JavaScript**
- **Flexbox та CSS Grid** для адаптивної верстки
- **Google Fonts** (Montserrat, Open Sans)
- **LocalStorage** для збереження теми
- **GitHub Pages** (для деплойменту)
